# TJP ホームページ

スマートフォン向けに最適化されたシンプルなホームページです。

## 特徴

✅ **モバイルファースト** - スマートフォンに完全対応  
✅ **高速** - シンプルなコードで高速読込  
✅ **ダークモード対応** - 自動的にシステム設定に対応  
✅ **タッチフレンドリー** - タップしやすいボタンサイズ  

## ファイル構成

```
.
├── index.html      # ホームページ本体
├── style.css       # スタイル
└── README.md       # このファイル
```

## GitHub Pagesで公開

1. リポジトリの **Settings** を開く
2. **Pages** セクションで **Source** を `main` に設定
3. 保存

**公開URL:** https://nonhoii.github.io/TJP.github.io/

## カスタマイズ

### テキストの変更
`index.html` の各セクションを編集してください。

### 色の変更
`style.css` で `#667eea` と `#764ba2` を好きな色に変更できます。

### セクションの追加
```html
<section id="new" class="section">
    <h2>新しいセクション</h2>
    <p>ここにコンテンツを入力</p>
</section>
```

## 動作確認

ローカルで確認するには、`index.html` をブラウザで開くだけです。
